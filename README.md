# Modelo de Previsão no Azure Machine Learning

Instruções passo a passo para criar e implantar um modelo de previsão usando o Azure Machine Learning.


## Passos

### 1. Criado um Workspace no Azure Machine Learning

- Acesso [Azure Portal](https://portal.azure.com/).
- Criado um serviço do [Azure Machine Learning](https://azure.microsoft.com/en-us/services/machine-learning/).
- Criado um novo espaço de trabalho do Azure Machine Learning.

### 2. Desenvolvimeto e Treinamento do Modelo

- Desenvolvido o modelo usando a biblioteca de machine learning.
- Utilizado dados apropriados para treinar o modelo.

### 3. Registrando o Modelo no Azure Machine Learning

- Registrado o modelo no Azure Machine Learning para rastreamento e gerenciamento.

### 4. Configuração do Ponto de Extremidade

- Configurado o ponto de extremidade para fornecer uma interface para interagir com o modelo.

### 5. Implantação do Modelo

- Implantado o modelo no Azure Machine Learning.

### 6. Pontos de Extremidades

- Pontos de extremidades configurados para fazer previsões com novos dados.
